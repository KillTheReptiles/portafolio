NETBY

This project helps the user to know their different network protocols, where they can find different network configurations such as their internet speed, the speed at which their device navigates. These parameters are displayed from a website and are shared between two types of users, the administrator user and the client user. The administrator user will be able to have access to all the users as well as being able to modify parameters, network configurations of each user. He will also be able to delete users and create them. The website is an API created with Node.js where you can communicate with a database already modeled and programmed to host all the data of each user. This website is intuitive, where users have a greater facility when witnessing the statistics and network configurations in their user, for this its interface (front-end) is made in HTML, CSS and Handlebars, so that it is a site modern, stable, robust, and interactive.

  • Category: Networks, Web development, Software Engineering
